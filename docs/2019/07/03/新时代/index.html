新时代

我的眼前突然亮了起来，模模糊糊听到有个人在说话。“李先生，醒一醒。”
我立刻睁开了眼，却没有意料之中的刺激，因为光线突然变暗了。“李先生，你终于醒了。”一个男子站在我床边说到。
“嗯。”我在他搀扶下坐了起来，但是还是有点迷糊。
